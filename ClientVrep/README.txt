Desafio: Levar o robô até o objetivo (área em verde, preferencialmente no quadrado laranja) utilizando as linguagens e ferramentas que preferirem (Remote Api, ROS, etc) para comunicação do seu programa com o simulador e o robô. Fica livre o uso de quaisquer sensores.
Boa sorte a todos!

Para quem for usar a Remote Api (recomendado), baixar a versão 3.4 rev 1 e usar cena goToGoal_remoteApi:
http://www.coppeliarobotics.com/downloads.html

Para quem quiser utilizar o ROS com o pacote RosInterface, usar a cena goToGoal_ROS e a versão 3.4 rev 17 do V-REP:
